Difficulté : moyen

Description :

L’usine du Supercalculateur était censée être libre d’accès : le vieux système de badges était désactivé depuis longtemps. Jusqu’à ce que X.A.N.A. le réactive, verrouillant à nouveau toutes les portes.

Chaque badge est lu par une caméra, qui utilise un modèle d’amélioration et de débruitage intégré pour interpréter correctement les images.

Jérémie a réussi à récupérer ce modèle, mais pas le système complet.

Aelita se souvient d’un détail crucial : le badge permettant l’accès a un fond rouge.

Pour entrer il va falloir trouver à quoi ressemble le badge qui dévérouille l'usine.
